# P1 效果受限原因：客观逐项排查

基于各批次 run（V2/V3/V4/V5）、全场景线性/非线性表、数据构成与留一数据集出结果，对「目前效果为什么不够好」做逐项排查。结论会区分：**数据与金标准**、**VAE 与表示**、**回归与评估设计**、**跨数据集与样本量**。

---

## 一、数据与金标准

### 1.1 样本量小，且金标准分布不均

- **有金标准 subject 数**：全量 D1+D2+D4 约 127（V4 latent CSV：D1=59, D2=38, D4=30）；**仅 D1+D2 评估时 n=63**。
- **5-fold 按 subject 划分**：每折约 12–13 个 test subject，极易受少数点影响，Spearman r 折间波动大（如 V4 主 run：SSPG r 单折 0.49–0.78，DI r 0.20–0.64）。
- **结论**：样本量是硬约束；当前 r≈0.5–0.65 在 n≈63 下已属中等相关，再提升需要更多有金标准样本或更稳的评估方式（如重复 5-fold、报告 CI）。

### 1.2 D4 与 D1/D2 金标准不在同一尺度

- **D4 的 DI**：绝大多数在 **0.01–0.15**，而 D1/D2 常见 **0.4–6.6**；D4 的 DI 方差极小，且与 D1/D2 几乎不重叠。
- **D4 的 SSPG**：范围与 D1/D2 有重叠，但 subject 少（约 20+），在图上呈一小团。
- **文档已指出**：D4 金标准很可能来自不同人群/协议/定义（如 Hall 2018 自由生活 CGM，若 DI 为后配则单位或公式可能不一致）。
- **证据**：  
  - V2 在「全数据（含 D4）」上 DI r 可达 **0.71**，但**仅 D1+D2 评估时 DI r 掉到 0.38**，说明高 DI 主要来自 D4 那一段「易拟合」的窄带。  
  - 留一数据集出在 **D4 上**（V3 等）常出现 RMSE/R² 崩掉（如 RMSE 数百、R² 极负），说明用 D1+D2 训的 Ridge 在 D4 上外推失效。
- **结论**：**D4 与 D1/D2 不是同一把尺子**；把 D4 混入训练或评估会「虚高」全数据 r，或导致外推崩溃。主结果只用 D1+D2 评估是合理选择；D4 更适合作为单独分析或表型探索。

### 1.3 单潜变量（si/mi）与金标准相关性弱

- V4 主 run、**全量 63 人**上：si→SSPG r≈0.14，mi→SSPG r≈0.12；si→DI r≈0.08，mi→DI r≈0.08（见 evaluation 全量 direct_*）。
- **6D Ridge 明显优于单 si/mi**：SSPG 从 ~0.12 提到 0.63，DI 从 ~0.08 提到 0.39（D1+D2 5-fold）。说明**单维不足以预测**，需要多维联合或非线性。
- **结论**：金标准与「单一」潜变量关系弱，不是 bug，而是生理上 SSPG/DI 依赖多参数；当前 6D Ridge 已经利用了这一点，但受样本量与尺度限制，r 仍有上限。

## 二、VAE 与表示

### 2.1 VAE 拟合质量未系统验证

- 当前**未在每次 run 后保存**训练曲线、验证重建 MSE、示例 CGM；因此无法从现有结果里直接判断「VAE 是否拟合充分」。
- 若 VAE 重建差，潜变量会掺杂大量噪声，下游 Ridge 的 r 上限会被压低。
- **建议**：重跑至少一个 run（或在新 run 中）保存 `training_curves.json`、`reconstruction_val_mse.npy`、`reconstruction_examples.npz`，用 `assess_vae_fit.py` 出图；先确认「拟合是好的」，再谈表示与回归。

### 2.2 潜空间是否「对齐」金标准

- **6D 联合** 明显优于单 si/mi，说明 6D 里确实含有与 SSPG/DI 相关的信息。
- 但 **R² 仍偏低甚至为负**（如 V4 主 run DI R²≈-0.21）：说明线性 Ridge 在 6D 上只能解释很少的方差，或存在非线性/异方差。
- **含 D4 时**：线性 vs 非线性表中，**DI 上 poly2 优于线性**（V3 全数据：poly2 r≈0.70 vs linear 0.63），说明在「跨尺度」数据上，潜变量–金标准关系更接近非线性。
- **结论**：表示里**有**信息，但（1）线性映射可能不够；（2）若含 D4，尺度混杂会进一步凸显非线性。主结果用 D1+D2 + 线性已是最稳组合；若纳入 D4 或追求更高 r，可报告 poly2/GB 或加敏感性分析。

### 2.3 不同 seed/λ 下潜空间差异大

- 同一套数据，**不同 run（seed × LAMBDA_IR）** 的 5-fold r 差异很大：例如 V4 中 SSPG r 从 0.21 到 0.65，DI r 从 0.25 到 0.51。
- 留一数据集出在**部分 run** 上会崩（如某 run 在 D2 上 SSPG RMSE 520、homa_b RMSE 282），说明某些初始化/超参下学到的潜空间在域间外推极差。
- **结论**：**VAE 训练不稳定**；选「SSPG_r + DI_r 之和」最大的 run 作为主结果是合理的，但需在文中说明存在 seed 敏感性，并尽量报告区间或多 seed 平均。

## 三、回归与评估设计

### 3.1 线性 Ridge 的局限

- 已做对比：在**仅 D1+D2** 上，线性 Ridge 与 log-target、poly2、GB 相比**并不差**，有时略优（SSPG 上线性 r≈0.63）。
- 在**含 D4** 时，**DI 上 poly2 更优**（r≈0.70），说明线性不是「错」，而是**在跨尺度数据上不够灵活**。
- **结论**：主结果用线性是可接受的；若审稿人质疑，可补充「含 D4 时 DI 用 poly2 的敏感性分析」或直接在主表旁加一列 poly2。

### 3.2 评估口径不一致会显得「效果差」

- **全数据（D1+D2+D4）** 上 5-fold r 通常**低于**「仅 D1+D2」上的 r（如 V4：全数据 SSPG r≈0.50，仅 D1+D2 则 0.63），因为 D4 的尺度与 D1/D2 不一致，拉低整体相关。
- **选优标准**：V4 用「仅 D1+D2 评估」选优，所以主 run 的 r 是在 D1+D2 上报告的，与「全数据 r」不可直接比。
- **结论**：效果「好/差」要看口径；当前主结果（V4，D1+D2，SSPG r≈0.65，DI r≈0.51）是**在一致、可比口径下**的最优，不必与 V2 的「全数据 DI r=0.75」直接对比。

### 3.3 5-fold 折间方差大

- 单折 r 可高至 0.78（SSPG）或 0.64（DI），也可低至 0.49 / 0.20；**平均 r 受折划分影响大**。
- **结论**：报告 5-fold 均值的同时，建议补充**标准差或分位数**（已有 evaluation_metrics 中的 std）；若篇幅允许，可做多 seed 5-fold（如 3×5-fold）取平均，使估计更稳。

## 四、跨数据集与训练策略

### 4.1 仅 D1+D2 训练（V5）未带来提升

- V5（`P1_TRAIN_DATASETS=D1,D2`，强 SSPG/DI 头）：SSPG r≈0.44，DI r≈0.47，**均低于 V4**（D1+D2+D4 训练、D1+D2 评估）。
- **结论**：在当前数据量下，「更多数据（含 D4）训练 + 仅 D1+D2 评估」优于「仅 D1+D2 训练」；D4 在训练中提供了正则或多样性，而不是单纯噪声。

### 4.2 留一数据集出（D1 vs D2）表现尚可但不稳

- V4 主 run：D1 作 test 时 SSPG r≈0.62、DI r≈0.52；D2 作 test 时 SSPG r≈0.51、DI r≈0.46。说明**域间有一定泛化**，但 D2 略弱。
- 部分 run 在「D2 作 test」时崩掉（RMSE 极大、r 负），说明**潜空间对 seed/超参敏感**，域间外推不稳定。
- **结论**：留一数据集出可作为泛化性指标，但需配合「选优 run」或多 run 汇总；单 run 崩掉不代表方法无效，而是需报告稳健 run 的结果并说明敏感性。

### 4.3 D3 无法参与金标准评估

- D3 配置为 `has_sspg=False, has_di=False`，**无 SSPG/DI 金标准**，不能算预测 r/RMSE。
- **结论**：D3 只能用于表型/聚类或 encode 分布分析；「把 D3 纳入」在**预测金标准**层面目前无法实现，除非后续补标或改用其他任务。

---

## 五、总结：原因排序与建议

按「对当前数字影响大小」的粗略排序：

1. **数据尺度与组成（D4 vs D1/D2）**  
   D4 金标准与 D1/D2 不同尺度/定义，混用会虚高或崩掉；**主结果仅用 D1+D2 评估**已规避，但限制了「全数据 r」的解读。

2. **样本量小（n≈63）**  
   5-fold 折间方差大，单变量相关弱；**6D Ridge 已利用多维**，r≈0.5–0.65 在同类小样本研究中属常见范围；进一步提升需更多有金标准样本。

3. **VAE 训练与拟合未系统验证**  
   无法从现有 run 判断拟合是否充分；**建议补跑并保存训练/重建指标**，确认「先拟合好再谈预测」。

4. **训练/评估的 seed 与超参敏感**  
   不同 run 间 r 差异大，留一数据集出在部分 run 上崩；**选优 run + 报告方差/多 seed** 可提高可信度。

5. **回归形式（线性 vs 非线性）**  
   在仅 D1+D2 时线性足够；**含 D4 时 DI 用 poly2 更合适**，可作为补充或敏感性分析。

6. **单潜变量与金标准关系弱**  
   属生理与模型设计使然；**6D 联合已明显优于单维**，无需归为「效果差」的主因。

**建议的下一步（可选）**：  
- 补一次 VAE 拟合诊断（训练曲线 + 重建 MSE + 示例 CGM）；  
- 在主表或附录中报告「含 D4 时 DI 用 poly2 的 r」；  
- 在文中明确写出「主结果仅 D1+D2、n=63」「5-fold 均值±标准差」「部分 run 留一数据集出存在不稳定」，以增强可重复性与解释清晰度。
